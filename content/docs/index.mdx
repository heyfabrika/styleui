# Welcome to StyleUI

**Ship Faster, Stay in Style**

StyleUI is a premium component library that provides beautiful, production-ready UI elements for your Next.js projects. Built on modern web technologies, StyleUI helps you build stunning interfaces faster without compromising on design or functionality.

## What is StyleUI?

StyleUI is a component library—not a framework or a full-stack solution. It's a collection of carefully crafted UI elements that you can copy directly into your project and customize to fit your needs.

### Key Characteristics

- **Copy, Don't Import** - Components are copied into your codebase, giving you full ownership
- **Zero Dependencies** - No external packages to manage or version conflicts
- **Fully Customizable** - Modify any component to match your design system
- **Production Ready** - Battle-tested components optimized for performance and accessibility

## What StyleUI Offers

StyleUI provides three types of UI elements:

### 🧩 Components
Individual, reusable UI components like buttons, animated lists, counters, and more. These are the building blocks of your interface.

### 🏗️ Blocks
Pre-built sections and layouts like pricing tables, testimonials, team sections, and logo clouds. These are complete sections ready to integrate into your pages.

### 📄 Templates
Full page templates that combine multiple blocks and components into complete, ready-to-use page layouts. Perfect for landing pages, dashboards, and more.

## Tech Stack

StyleUI components are built with:

- **Next.js 16** - React framework with App Router
- **React 19** - Latest React features
- **TypeScript** - Full type safety
- **Tailwind CSS 4** - Utility-first styling
- **Motion** - Smooth animations
- **shadcn/ui** - Component architecture foundation

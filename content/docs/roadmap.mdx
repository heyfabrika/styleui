# Roadmap

Here's what we're planning for StyleUI. This roadmap is subject to change as we gather feedback and prioritize features.

## Coming Soon

### More Components
We're continuously adding new components to expand the library. Upcoming additions include:

- Form components with validation
- Data display components (tables, lists)
- Navigation components
- Feedback components (alerts, toasts, modals)

### More Blocks
New block patterns are in development:

- FAQ sections
- Contact forms
- Blog post layouts
- Dashboard widgets
- E-commerce sections

### More Templates
Additional templates are planned:

- Dashboard templates
- Blog templates
- E-commerce templates
- Documentation templates
- Portfolio templates

### Enhanced Features

- **Component Variants** - More styling options and variants
- **Animation Presets** - Pre-configured animation patterns
- **Theme Builder** - Visual theme customization tool
- **Component Playground** - Interactive component testing

## In Progress

- Expanding the component library
- Improving documentation
- Adding more examples
- Performance optimizations

## Future Considerations

- Component marketplace
- Design system integration
- Figma plugin
- CLI tools for faster setup

## Contributing

Have ideas for new components, blocks, or templates? We'd love to hear from you! Check out our [GitHub repository](https://github.com/heyfabrika/styleui) to contribute or suggest features.

## Stay Updated

Follow our progress and get updates on new releases by:

- Starring our [GitHub repository](https://github.com/heyfabrika/styleui)
- Checking back regularly for updates
- Following our announcements

---

*This roadmap is a living document and may change based on community feedback and priorities.*

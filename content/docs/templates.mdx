# Templates

Templates are complete, ready-to-use page layouts that combine multiple blocks and components into a cohesive design. They're perfect for getting started quickly with a professional-looking page.

## What are Templates?

Templates are full page layouts that include:

- **Multiple Blocks** - Pre-arranged sections like hero, features, pricing, testimonials
- **Navigation** - Complete navigation components
- **Layout Structure** - Proper spacing, responsive design, and flow
- **Theme Support** - Built-in light and dark mode support
- **Animations** - Smooth transitions and interactions

### Example: Notio Template

The Notio template is a modern landing page that includes:

- Hero section with call-to-action
- Logo cloud section
- Features showcase
- Statistics section
- Team member section
- Testimonials
- Pricing table
- Footer

All sections are pre-styled and work together to create a cohesive, professional landing page.

## Templates vs Blocks vs Components

### Components
Individual UI elements (buttons, cards, inputs)

### Blocks
Complete sections (pricing, testimonials, team)

### Templates
Full page layouts (landing pages, dashboards, complete designs)

## How Templates Work

### Structure

Templates are organized with:

- **Page Layout** - Main page structure and routing
- **Template-Specific Components** - Custom components for the template
- **Block Integration** - Reusable blocks integrated into the layout
- **Configuration** - Theme and styling configuration

### Usage

Templates can be used as:

1. **Starting Point** - Copy the template and customize it for your needs
2. **Reference** - Study the structure and patterns for your own pages
3. **Inspiration** - Get ideas for layouts and design patterns

## Customization

Templates are fully customizable:

- Modify the layout and structure
- Replace or remove sections
- Adjust colors and styling
- Add your own content
- Integrate with your backend

## When to Use Templates

Use templates when you need:

- **Quick Launch** - Get a complete page up and running fast
- **Professional Design** - Start with a polished, production-ready design
- **Learning** - Understand how to structure complex pages
- **Consistency** - Maintain design consistency across your site
